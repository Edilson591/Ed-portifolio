Sobre Mim

Técnico em radiologia migrando para desenvolvimento front-end, aplicando habilidades analíticas para criar interfaces web intuitivas e atraentes. Comprometido com aprendizado contínuo e contribuição para equipes de desenvolvimento, buscando experiências digitais de alta qualidade. Domínio de HTML, CSS e JavaScript, com experiência na construção de experiências digitais envolventes. Explorando React, TypeScript, Vue.js, Bootstrap e Angular. Animado para colaborar em equipes de desenvolvimento front-end e impulsionar a inovação no desenvolvimento web.
