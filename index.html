Agenda de Números

Contador para aniversario

Este projeto é um contador interativo que calcula e exibe o tempo restante até o próximo aniversário de uma pessoa. A aplicação utiliza JavaScript para manipulação de datas e proporciona uma contagem regressiva precisa, levando em consideração situações onde o próximo aniversário cai no ano seguinte.
